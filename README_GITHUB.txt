
# Mr. Black Gloves — Freecuts (GitHub Pages)

## Quick Deploy
1. Create a GitHub repo named `mbg-freecuts` (or any name).
2. Upload everything in this folder (not the ZIP) to the repo.
3. Go to **Settings → Pages → Build and deployment**.
   - Source: **Deploy from a branch**
   - Branch: **main** (root)
4. Wait a minute → your site is live at:
   `https://YOUR_GITHUB_USERNAME.github.io/mbg-freecuts/freecuts/`

## Custom Domain
- In Settings → Pages, set **Custom domain** to `mrblackgloves.com`
- Add a `CNAME` DNS record at GoDaddy pointing to `YOUR_GITHUB_USERNAME.github.io`

## Edit the QR
- Update `/freecuts/assets/qr_mbg_freecuts.png` later to match your final domain/URL.
